General purpose interactive programmer's tool for creating large blocks of similar code from list(s) of variables. It can create stupid code faster than you!



  • no config files, no script language
  • smart pattern matching [Case Inference]
  • free and open-source [MIT license]
  • cross-platform
  • (requires Java 6 or newer to run)
